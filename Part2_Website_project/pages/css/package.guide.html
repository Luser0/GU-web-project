Package.CSS Framework

Professional Bootstrap-like CSS Framework for EventsX Project

Getting Started

Package.CSS is a lightweight, variable-driven CSS framework designed for the EventsX web application. It provides consistent styling, responsive design, and easy customization through CSS variables.

Quick Start

Include the framework in your HTML file:

<link rel="stylesheet" href="/pages/css/package.css">

Features

CSS Variables

All colors, spacing, and sizing use CSS variables for easy theme customization.

Responsive Design

Built-in responsive utilities and mobile-first approach for all devices.

Flexible Components

Ready-to-use buttons, forms, cards, alerts, and more.

Easy Theming

Change colors and spacing by modifying CSS variables in :root.

Mobile Tips

  • Always include viewport meta tag: <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Test on multiple devices
  • Use flexbox for layouts instead of fixed widths
  • Buttons and form inputs scale automatically

Best Practices

Follow these guidelines for the best results when using Package.CSS:

✓ Do's

  • Use semantic HTML elements (header, nav, main, section, footer)
  • Use CSS variables for customization instead of hardcoding values
  • Leverage utility classes for spacing and alignment
  • Test on multiple devices and browsers
  • Use the .container class for content wrapping
  • Apply meaningful button variants (primary, secondary, success, danger)
  • Keep form labels associated with inputs using for attribute

✗ Don'ts

  • Don't use inline styles when utility classes are available
  • Don't mix custom CSS with framework classes without testing
  • Don't use hard-coded colors; use CSS variables instead
  • Don't forget to include the package.css link tag
  • Don't disable browser default focus styles without replacing them
  • Don't nest too many flexbox containers

Customization Example

To change the entire theme, modify the CSS variables in package.css: 

:root {
    /* Primary brand color */
    --primary: #ff6b6b;
    --primary-dark: #ee5a52;
    
    /* Success color */
    --success: #4ecdc4;
    
    /* Font family */
    --font-heading: 'Georgia', serif;
    
    /* Spacing scale */
    --spacing-md: 20px;
}
